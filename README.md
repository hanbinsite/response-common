
# 公共仓返回，定义公共code，msg部分
## 说明
1. 用于项目基础使用，公共返回类
2. 使用简单

## 使用
~~~ xml
<dependency>
    <groupId>site.hanbin</groupId>
    <artifactId>response-common</artifactId>
    <version>0.0.1</version>
</dependency>
~~~
